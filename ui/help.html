Post Occupancy Monitor

The module periodically polls GET /api/crm/post-states and saves an occupancy snapshot to data/last_snapshot.json.

“Busy” logic

  • Post is free when online and mode is program_9 (payment entry).
  • Otherwise, when connected — busy.

Settings

  • poll_interval — API poll interval (seconds, 15–3600).

Start

Requires PyOrchestrator. After install, click Start on the module card.
