Монитор занятости постов

Модуль периодически опрашивает GET /api/crm/post-states и сохраняет снимок занятости в data/last_snapshot.json.

Логика «занят»

  • Пост свободен, если онлайн и режим program_9 (внесение оплаты).
  • Иначе при активном подключении — занят.

Настройки

  • poll_interval — интервал опроса API (секунды, 15–3600).

Запуск

Требуется PyOrchestrator. После установки нажмите Запустить на карточке модуля.
